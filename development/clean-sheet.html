Dark Theme with FlatScrollbars

For fans of the Dark theme on Windows, the Code Affine Theme feature adapts the Xiliary FlatScrollBar overlay on supported Scrollable types.

Theme: Dark

Default Windows Theme with FlatScrollbars

For fans of the default Windows theme, the Code Affine Theme feature provides a derivation that incorporates the Xiliary FlatScrollBar component on supported Scrollable types.

Theme: Windows [Flat-Scrollbar]

Note

The FlatScrollbar overlay mechanism used by the Windows 10 Clean Sheet variant is currently only available for table, tree and styled text based UI components. Due to the variety of the Eclipse ecosystem it might not work properly in all scenarios. Issue and usability reports are highly appreciated to improve the user experience.
